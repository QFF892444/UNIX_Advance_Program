1.3 因为 perror 的 msg 参数是一个指针， perror 就可以改变 msg 指向的字符串。然而使用限定符 const 限制了 perror 不能修改 msg 指针指向的字符串。而对于 strerror, 起错误号参数是整数类型，并且 C 是按值传递所有参数，因此即使 strerror 函数想修改参数的值也修改不了，也就没有必要使用 const 属性。
